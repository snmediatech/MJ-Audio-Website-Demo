# MJ Audio — The Real Cinema - Product Requirements Document

## Project Overview
Professional service-provider website for MJ Audio, a premium home theater design, installation, and calibration specialist in Tirunelveli, Tamil Nadu. Complete redesign based on Surroundswar reference website structure, with focus on storytelling, project showcase, and service-oriented positioning.

---

## Original Problem Statement
Build a modern, professional website for **MJ Audio — The Real Cinema**, positioning them as a premium home theater & audio-video solutions provider (NOT a reseller) specializing in design, installation, calibration, and complete cinema experiences.
